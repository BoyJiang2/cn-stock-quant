# 开发路线

## v0.1：研究回测闭环

目标：可以在本地同步日线数据，运行双均线策略，看到回测指标和交易记录。

已完成骨架：

- FastAPI 后端
- SQLite 数据库模型
- AkShare 股票列表和日线适配
- 策略注册表
- 双均线策略
- 日频回测引擎
- 回测结果持久化
- 历史回测查询
- 风控引擎骨架
- 交易计划骨架
- React 控制台页面
- 净值曲线和回撤曲线
- 基础 Markdown 文档

还需要补充：

- 安装依赖后做真实 AkShare 同步测试
- 数据同步日志
- 更明确的错误提示

## v0.2：数据中心增强

目标：让数据可持续维护，而不是每次手工调用接口。

任务：

- 交易日历
- 指数行情
- 多股票批量同步
- 增量同步
- 数据完整性检查
- 数据同步状态页
- SQLite 索引优化

验收：

- 可以选择股票池并批量同步最近 N 年日线。
- 能看到每只股票最新数据日期。

## v0.3：策略和因子框架

目标：支持多个策略和可配置参数。

任务：

- 策略配置保存
- 参数表单
- 股票池过滤器
- 动量策略
- 低估值加动量策略
- 因子计算缓存

验收：

- 前端可以选择策略和参数运行回测。
- 策略文件结构足够清晰，后续新增策略不需要改回测引擎。

## v0.4：A 股真实规则

目标：让回测更接近实际交易。

任务：

- T+1
- 涨跌停处理
- 停牌处理
- ST 过滤
- 退市风险过滤
- 调仓日规则
- 成交价模型
- 基准指数对比

验收：

- 回测不会在涨停买入、跌停卖出、停牌交易。
- 回测报告展示基准超额收益。

## v0.5：模拟盘和交易计划

目标：从研究工具进入日常交易辅助。

任务：

- 模拟账户
- 模拟持仓
- 每日目标仓位
- 交易计划生成
- 模拟成交
- 账户净值曲线

验收：

- 每天同步数据后，可以生成次日交易建议。
- 交易计划包含代码、名称、方向、股数、参考价和预计金额。

## v0.6：风控系统

目标：策略之外增加独立保护层。

任务：

- 单票仓位限制
- 总仓位限制
- 最大持仓数量
- 黑名单
- 行业集中度限制
- 单日亏损阈值
- 回撤阈值

验收：

- 策略输出目标仓位后，风控可以调整或拒绝订单。
- 所有拒绝原因都可追踪。

## v1.0：稳定个人版

目标：形成可长期使用的本地个人量化系统。

任务：

- Docker Compose
- 备份和恢复
- 日志和告警
- 权限保护
- 半自动实盘接口设计
- 使用手册

验收：

- 数据同步、回测、模拟盘和交易计划稳定。
- 实盘交易前只需要适配具体券商 broker。
